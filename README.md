# 🎷 Lazarin Luthier - Fine Wind Instruments

![License](https://img.shields.io/badge/license-MIT-gold)

> **Onde a precisão técnica encontra a alma da música.** > Site institucional e portfólio da Lazarin Luthier, focado na manutenção e restauração de instrumentos de sopro.

---

## 💎 O Projeto

Este site foi desenvolvido para ser a vitrine digital da **Lazarin Luthier**. O objetivo é transmitir a sofisticação e o cuidado artesanal aplicados em cada saxofone, flauta ou clarinete que passa pela bancada, unindo uma estética premium com tecnologias modernas de desenvolvimento.

---

## 🛠️ Stack Tecnológica

Para garantir performance e uma manutenção de código limpa, o projeto utiliza:

* **React + Vite**: Para uma interface rápida e reativa.
* **TypeScript**: Tipagem estática para evitar erros em tempo de execução.
* **Sass (SCSS)**: Pré-processador para estilos avançados.
* **Metodologia BEM**: Organização de classes CSS (`block__element--modifier`).
* **Lucide Icons**: Conjunto de ícones minimalistas.

---

## 🎨 Identidade Visual

A interface foi projetada com uma paleta de cores que remete ao luxo e à tecnologia:
* **Navy Blue**: Base sólida e profissional.
* **Gold**: Destaque para a elegância e o metal dos instrumentos.
* **Violet Accent**: Toque de modernidade e criatividade.

---

## 📂 Como Executar

Certifique-se de ter o **Node.js** instalado em sua máquina.

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/seu-usuario/lazarin-luthier.git](https://github.com/seu-usuario/lazarin-luthier.git)
    ```

2.  **Instale as dependências:**
    ```bash
    npm install
    ```

3.  **Rode o projeto em modo de desenvolvimento:**
    ```bash
    npm run dev
    ```
